# Hipnote

Post a notification in hipchat

## Installation

Add this line to your application's Gemfile:

    gem 'hipnote'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hipnote

## Usage

Make sure you assign `ENV['HIPCHAT_API_TOKEN']`
